# Verified Filter

We broke up the verification of the core Katydid algorithm into several folders.

* [Grammar](./Grammar/Readme.md): Regular Hedge Grammar's definitions and proofs.
* [Regex](./Regex/Readme.md): Regular expression's definitions and proofs.
* [Parser](./Parser/Readme.md): The Parser class and an implementation of a Hedge Parser.
* [Related](./Related/Readme.md): Experiments with Related work, RelaxNG.
* [Std](./Std/Readme.md): Definitions and proofs that we might expect to be in a standard library.
* [Pred](./Pred/Readme.md): Example symbolic predicate types that are used by both `Grammar` and `Regex`.
* [ExampleSmallBlogpost.lean](./ExampleSmallBlogpost/ExampleSmallBlogpost.lean): Example blog post as a regular hedge grammar in Lean.
